# Guía: Cómo compartir el Visor Estudiantil

Esta guía explica cómo distribuir la aplicación para que otros profesores puedan utilizarla en sus propias computadoras o mediante la nube.

## Opción A: Compartir por OneDrive / Google Drive (Recomendado)
Esta es la mejor opción si quieres que todos tengan la misma estructura de fotos.

1. **Crear Carpeta Compartida**: Crea una carpeta llamada `VisorEstudiantil` en tu Drive/OneDrive.
2. **Subir Archivos**: Copia el archivo `index.html` y todas tus carpetas de niveles dentro de ella.
3. **Compartir Acceso**: Envía una invitación de "Editor" a los otros profesores.
4. **Uso**: Cada profesor debe asegurarse de que la carpeta esté sincronizada en su PC. Al abrir el `index.html`, seleccionarán esa carpeta local sincronizada.

## Opción B: Uso de Hosting Gratuito (Netlify)
Para que los profesores no tengan que tener el archivo `index.html`, solo el enlace.

1. Entra a [Netlify](https://www.netlify.com/).
2. Sube solo el archivo `index.html`.
3. Envía el link generado (ej: `https://visor-centro-educativo.netlify.app`) a los profesores.
4. **Nota**: El profesor siempre deberá seleccionar su carpeta local de fotos al entrar, por seguridad del navegador.

## Sugerencias para los Profesores
Para que el sistema funcione perfecto, diles que sigan estrictamente el formato de nombre de archivo:
`Cédula - Nombre Completo.jpg`
